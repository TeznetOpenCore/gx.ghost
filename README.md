# gx.ghost
A mobile lightweight commz device
Decentralized user-tech continues with this project. With ESP's and Lora, a new mobile device is under development. With a keypad and oled screen, all the parts can be found online.

The Printer, is a base script allowing the user to hard code their own unique Commands into the Lora and have them be sent
 
 Rexxy256 is a mini-game where you stack bits without the enemy knocking them down
 PrimeBoss is a mini-game where you try to find as many Prime Numbers before the time runs out
